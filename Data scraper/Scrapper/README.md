# tutiempo_crawler

a Scrapper for climate data on en.tutiempo.net

it works fine now(2019-11-28), and contact me if you have any questions.

# quick start 

1. install python3

2. `pip install -r requirements.txt`

3. `vi demo.py` edit year & month & climatic station id

4. `ptyhon demo.py`
